# Heart Disease Data Analysis

This project analyzes the Heart Disease UCI dataset to explore patterns and relationships between clinical attributes and heart disease diagnosis. The dataset contains 1025 patient records with 14 clinical features including age, cholesterol, blood pressure, and heart rate.

**Dataset:** Heart Disease UCI Dataset (`heart.csv`) - [Source](https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset)

## How to Run the Project

### 1. Install Dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the Notebook

Open and run the Jupyter Notebook:

```bash
jupyter notebook data_workflow.ipynb
```

Run all cells in order from top to bottom. The notebook will:
- Load and display the dataset
- Clean the data (remove duplicates, rename columns)
- Perform exploratory data analysis
- Generate three visualizations
- Present a summary of findings
